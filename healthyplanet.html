2017 - Freelance
Healthy Planet

Home / Portfolio

Brought back by Healthy Planet in a freelance capacity (2017) to help redevelop the e-commerce site built on Magento with a new UI. Designs had been created and provided by the client - I was responsible for setting up the processes and tools for their in-house team of Web Designers. Tasks included creating HandlebarsJS templates for the numerous pages and use cases on the site, creating a Git repository while acting as admin for code reviews and approving/merging pull requests, setting up a Kanban Board on Trello for QA tickets as well as other miscellaneous tasks and support as required. Consequently, I set up a Slack channel with all respective plugins integrated in order to maintain communication and for general monitoring.

Tools/Techniques Used
- HandlebarsJS
- OWL Carousel
- Font Awesome
- Trello
- Slack
